# cmsplugin-slick-gallery


### Requirements

djangocms-text-ckeditor

slick.min.js

slick-theme.min.css

slick.min.css


### Installation

Add cmsplugin_slick_gallery to INSTALLED APPS:

```python
INSTALLED_APPS = (
        ...
        'cmsplugin_slick_gallery',
        ...
    )
```

Remember to add js/css requirements to `<head>` in your base template
